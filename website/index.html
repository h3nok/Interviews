Example: Self-Attention Equation

The core of the Transformer is the self-attention mechanism, mathematically defined as:

$$\text{Attention}(Q, K, V) = \mathrm{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$ 

# PyTorch implementation of scaled dot-product attention
scores = torch.matmul(Q, K.transpose(-2, -1)) / math.sqrt(d_k)
weights = F.softmax(scores, dim=-1)
output = torch.matmul(weights, V)
